# Spotify Plugin for StatusBar

Now playing widget for [StatusBar](https://github.com/hytfjwr/StatusBar).

## Features

- Current track info display (title, artist)
- Event-driven updates via Spotify notifications

## Install

In StatusBar preferences → Plugins → Add Plugin:

```
hytfjwr/statusbar-plugin-spotify
```

## Development

```bash
make build      # Release build
make dev        # Build & install locally
make release    # Build & publish GitHub Release
```

## Requirements

- macOS 26+
- [StatusBar](https://github.com/hytfjwr/StatusBar)
- Spotify
